Ubuntu 安装谷歌浏览

下载安装Google Chrome浏览 deb包

sudo dpkg -i google-chrome-stable_current_amd64.deb

安装依赖

sudo apt-get -f install libappindicator1 libindicator7

安装谷歌浏览器

sudo dpkg -i google-chrome-stable_current_amd64.deb

Ubuntu 搜狗输入法

下载下来deb包之后，直接

sudo dpkg -i sogoupinyin_2.1.0.0086_amd64.deb

如果遇到有依赖问题

sudo apt-get -f install

安装完之后，打开__Language Support__，将系统输入法设置为fcitx，然后重启/注销登录

登入系统后，配置fcitx，添加sogou输入法 配置完重启虚拟机即可正常使用

Go 语言环境安装

先去https://golang.org/官网下载Go 语言安装包

安装Go

以管理员的身份（sudo）将安装包（压缩包）解压到/usr/local 在压缩包所在的文件夹下打开终端（也可以 cd 到压缩包所在的文件夹）

在终端中输入命令sudo tar -C /usr/local -xzf go1.9.2.linux-amd64.t ar.gz

然后系统会提示您输入密码，密码就是开机密码 解压完成后，

在/usr/local路径下面有一个go文件夹 添加/usr/local/go/bin到环境变量PATH中。

通过vi命令将export PATH=$PATH:/usr/local/go/bin添加到$HOME/.profile 中

操作步骤如下：

  • vi $HOME/.profile
  • 收入i切换到编辑模式，插入export PATH=$PATH:/usr/local/go/bin
  # ~/.profile: executed by the command interpreter for login shells.
  # This file is not read by bash(1), if ~/.bash_profile or ~/.bash_login
  # exists.
  # see /usr/share/doc/bash/examples/startup-files for examples.
  # the files are located in the bash-doc package.
  # the default umask is set in /etc/profile; for setting the umask
  # for ssh logins, install and configure the libpam-umask package.
  #umask 022
  # if running bash
if [ -n "$BASH_VERSION" ]; then
# include .bashrc if it exists if [ -f "$HOME/.bashrc" ]; then
. "$HOME/.bashrc" fi
fi
  # set PATH so it includes user's private bin directories
  PATH="$HOME/bin:$HOME/.local/bin:$PATH"
  export PATH=$PATH:/usr/local/go/bin
  • 按esc键切换到命令行模式，再输入：，最后输入wq保存并退出
  • 执行 localhost:~ lin$ /usr/local$ source $HOME/.profile

测试安装：在终端中输入go version 检测版本号即可

安装编辑器ATOM

安装 gdebi

如果没有安装gdebi，请先安装

$ sudo apt-get install gdebi
$ gdebi --version 0.9.5.7ubuntu1
Atom下载并安装

进入官网https://atom.io/ 下载.deb文件 在压缩包所在的文件夹下打开终端，输入sudo gdebi atom-amd64.deb即可完成安装

测试安装：在终端中输入atom -version检测版本号

atom字体大小设置： 菜单->Edit->Preference->Editor->Font Size

atom目录字体大小 菜单->Edit->Stylesheet

  • 目录： 
    .tree-view {
  // background-color: whitesmoke;
  font-size: 28px
}
  • Tab ：
.tab-bar .tab {
    font-size: 16px;
}

  • go-plus
  • file-icons
